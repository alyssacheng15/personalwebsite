Personal Website Process

A look into the process I took to design and code this website, as well as the branding, colors, and typography.

role: coding in html/css, javascript, figma wireframing

view live github repo

branding.

wireframing with figma.

I used Figma to map out and wireframe a rough outline of my website and made some of the graphics in Canva.

testing the cursor.

With Javascript and CSS, I created an interactive animated cursor that expands when a link is hovered. I also developed numerous animations for buttons and links.

I experiemented with z-values, transitions, and transformations in CSS, and made functions and query selectors with Javascript.

coding the webpages.

At the end, I had over a dozen code files, and thousands of lines of code.

3-d modeling.

Using Spline, I modeled and surface a 3-D rendering of a workspace with all my favorite things — music, coding, and travel.

demo.

After dozens of hours of coding, modeling, and designing, I deployed my website using Netlify.
